# ACTIVE_GUARDS.md

> Operational Guard Register - LOG file
> Record recurring failures to prevent repetition

## Guard Format

```markdown
### GXXX: [Short Guard Name]
- **Trigger**: What situation activates this guard?
- **Prevention**: What to do instead?
- **First seen**: YYYY-MM-DD
- **Recurrence test**: How to verify this guard is working?
- **Last triggered**: YYYY-MM-DD (update when hit)
```
