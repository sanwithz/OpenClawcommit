## QuickChart QR Code Generator

Learned method for generating QR codes via QuickChart API:

**Base URL:** `https://quickchart.io/qr`

**Parameters:**
- `text` - The URL/text to encode
- `size` - Image size in pixels (e.g., 300)
- `ecLevel` - Error correction level (L/M/Q/H)
- `dark` - Dark color (default: black)
- `light` - Light color (default: white)

**Example:**
```
https://quickchart.io/qr?text=https://example.com&size=300
```

**Usage:**
```bash
curl -s "https://quickchart.io/qr?text=URL&size=300" -o qr.png
```
